## Compilação

Explicação dos diretórios:
* bin: Diretório do executável (Terá o nome cage).
* generated: Diretório que conterá os arquivos gerados pelo flex e yacc.

```
$ make
```

## Execução

Para executar:
```
$ ./bin/cage < inputs/input.txt
```
